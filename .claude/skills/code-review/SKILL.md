## Review Checklist

### Chemistry Code
- [ ] Every `Chem.MolFromSmiles()` call has a null check immediately after
- [ ] Tanimoto novelty check is present in Agent 4 output — threshold is 0.85
- [ ] SA Score threshold enforced — drop > 8.0, warn 6.0–8.0
- [ ] hERG and Ames hard-fail logic present in Agent 5
- [ ] No SMILES generated by Gemini — only RDKit/DeepChem
- [ ] SMILES validated before writing to `candidate_compounds` in state

### Agent Code
- [ ] Agent only writes to its designated state field (see field ownership table in `/agent-engine`)
- [ ] `status_updates` emitted at start and end of every agent
- [ ] All external HTTP calls use `httpx.AsyncClient`, not `requests`
- [ ] Retry logic present on all external API calls (3 attempts, exponential backoff)
- [ ] Exceptions caught and written to `state["errors"]`, not raised
- [ ] Agent returns full `state` object — not a partial dict, not None

### Orchestrator Code
- [ ] `graph.py` has been reviewed with `gitnexus_impact` before any edit
- [ ] `state.py` schema matches what agents actually read and write
- [ ] Parallel branches use correct LangGraph parallel edge syntax
- [ ] Graph is compiled with `.compile()` before use

### API Code
- [ ] All endpoints have Pydantic request/response models
- [ ] Error responses use `{"error": "...", "detail": "...", "agent": "..."}` format
- [ ] No hardcoded values — everything from `config.py` or env
- [ ] WebSocket manager cleans up disconnected clients

### Frontend Code
- [ ] No hardcoded hex values — Tailwind classes or CSS variables only
- [ ] No emojis as icons — lucide-react only
- [ ] WebSocket `useEffect` returns cleanup function (`ws.close()`)
- [ ] All three states handled: loading, error, empty
- [ ] Molecule SVG rendered as `<img>` with `className="invert"` for dark mode

### General
- [ ] No API keys in source code — only in `.env`
- [ ] No `print()` statements left in production code — use logging
- [ ] No `TODO` comments left without a GitHub issue reference
- [ ] All new functions have a one-line docstring

## Common Code Smells in This Project

```python
# SMELL: Synchronous HTTP in async agent
response = requests.get(url)  # blocks event loop

# FIX:
async with httpx.AsyncClient() as client:
    response = await client.get(url)
```

```python
# SMELL: No SMILES validation
candidates.append({"smiles": generated_smiles})

# FIX:
mol = Chem.MolFromSmiles(generated_smiles)
if mol is not None:
    candidates.append({"smiles": generated_smiles})
```

```python
# SMELL: Trusting Gemini JSON without error handling
result = json.loads(gemini_response.text)

# FIX: use parse_gemini_json() from agent-engine skill
try:
    result = parse_gemini_json(gemini_response.text)
except json.JSONDecodeError:
    state["errors"].append("Gemini returned malformed JSON")
    return state
```

```jsx
// SMELL: No WebSocket cleanup
useEffect(() => {
  const ws = new WebSocket(url);
  ws.onmessage = ...
}, []);

// FIX:
useEffect(() => {
  const ws = new WebSocket(url);
  ws.onmessage = ...
  return () => ws.close();
}, []);
```
